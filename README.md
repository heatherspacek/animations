## "MESS"?
The [Melee Empirical Strategy Solver (MESS)](https://github.com/heatherspacek/MESS) is my long-term project to make a human-reaction-time, human-useful, strategy engine (akin to a chess engine) for SSBM.

Even if it never comes to fruition as a full system, I expect the endeavour to produce interesting and useful research/labbing tools along the way *(...like this one!)*
